## 📁 文件夾結構

### 📂 `samples/` - 樣本與參考文件

#### `建物謄本PDF範例/`
**包含**: 房屋謄本的 PDF 樣本  
**用途**: 測試 AI 謄本解析功能  
**何時使用**: 
- 開發謄本提取功能
- 測試 OCR 和數據提取
- 驗證數據格式

---

### 📂 `data/` - 數據與參考表

#### `大安區地號建物名稱對照表.xlsx`
**包含**: 台北市大安區的地號與建物名稱對照  
**用途**: 房產數據查證和驗證  
**何時使用**:
- 驗證輸入的地址資訊
- 地號查詢和轉換
- 生成測試數據

## 📝 新增資源

如需新增資源文件，請按以下規則：

1. **確定文件類型**
   - 樣本/示例 → 放入 `samples/`
   - 參考/查詢表 → 放入 `data/`

2. **命名規則**
   - 使用中文描述性名稱
   - 例如：`新增資源名稱.xlsx`

3. **更新本文件**
   - 在上面的表格中添加新條目
