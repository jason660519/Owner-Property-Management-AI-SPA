# 專業詞彙對照表 - 台灣繁體中文 / 英文（資料庫版）

本文件為「房東物件管理語音 AI App」專案之專業詞彙對照表，已依下列 11 欄位順序重新編排，方便資料庫匯入與後續維護：

1. ID（唯一識別碼）
2. 分類1 (必填)
3. 分類2（可為空）
4. 分類3（可為空）
5. 專業名詞的英文縮寫(可為空)
6. 專業詞彙的英文名稱(必填)
7. 英文解釋(必填)
8. 專業詞彙的中文名稱(必填)
9. 中文解釋(必填)
10. 建立時間
11. 更新時間

---

## 資料表結構說明

```sql
CREATE TABLE glossary (
  id SERIAL PRIMARY KEY,
  category1 TEXT NOT NULL,
  category2 TEXT,
  category3 TEXT,
  term_abbr_en TEXT,
  term_en TEXT NOT NULL,
  description_en TEXT NOT NULL,
  term_zh TEXT NOT NULL,
  description_zh TEXT NOT NULL,
  created_at TIMESTAMPTZ DEFAULT NOW(),
  updated_at TIMESTAMPTZ DEFAULT NOW()
);
```

> 說明：
>
> - category1 以中文主分類為主，category2 為對應英文分類，category3 保留作為次階層或標籤，若無則填 "-"。
> - term_abbr_en 無縮寫時填 "-"。
> - description_en 目前以英文名稱作為簡短說明，確保非空；後續可再補充詳細解釋。
> - created_at / updated_at 皆使用 ISO 8601 UTC 時間字串示意，可由資料庫自動帶入。
