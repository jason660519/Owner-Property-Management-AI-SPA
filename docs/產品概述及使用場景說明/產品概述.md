# 房東物件管理語音 AI App - 產品概述

## 1. 產品願景

建立一個專門為房東設計的房屋管理 AI 語音 App，透過自然語言對話，協助房東高效率處理房屋**出租、出售與持有期間管理**及相關財稅問題，降低物業管理與租賃合約管理的複雜度。

## 2. 產品 Landing Page 與註冊流程

本產品僅提供給**實際不動產所有權人或其合法授權代理人（如代管公司、會計師事務所）**使用，所有帳號申請皆以房東身分為核心設計。

公司產品的 Landing Page 分為左右兩區塊：

- **右邊**：公司產品介紹與價值主張。
- **左邊**：會員註冊與登錄功能。

### 註冊機制

提供多種便捷註冊方式：

1.  **手機註冊**：手機號碼 + 手機驗證碼。
2.  **社群帳號註冊**：Google / Facebook 等主流社交軟體帳號 + 手機驗證碼驗證。
3.  **Email 註冊**：Email + 手機驗證碼驗證。

### 登入機制

- **帳號 (Email) + 手機驗證碼登入**，未來可擴充雙重驗證（2FA），以提高房東後台登入之安全性。
